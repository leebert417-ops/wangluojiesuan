# 项目文档（document）

本目录为 `wangluojiesuan` 项目的文档入口，面向“读者/使用者/二次开发者”，尽量把信息按 **理论 → 算法 → 用法** 组织起来。

项目包含两部分：
- **General Problem Solver**：通用通风网络求解器（App + 可脱离 App 使用）
- **Solver for fig5.2**：针对教材图 5.2 的专用求解器（更快、但拓扑固定）

> 若你只关心通用求解器，请优先阅读第 5/6 章（App 用法与 App 外用法）。
