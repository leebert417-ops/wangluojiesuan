# 01 简介

`wangluojiesuan` 是一套基于 MATLAB 的通风网络求解与可视化工具集，核心目标是：
- 给定通风网络的**拓扑（节点/巷道）**与**风阻**，在指定边界条件（入风/回风节点、总风量）下求解每条巷道的风量与压降；
- 支持图形界面（App Designer）交互，同时支持脚本方式批处理；
- 以经典 **Hardy Cross** 回路迭代作为核心数值方法，并自动识别基本回路。

项目由两部分组成：

## 1) 通用求解器（General Problem Solver）

适用：任意规模、任意拓扑的通风网络。

核心特性：
- 自动识别独立回路（基本回路），无需手工列回路矩阵；
- 支持多入风/多回风节点；
- 提供 App 界面与 CSV 导入/导出。

入口与代码位置：
- App：`General Problem Solver/ui/NetworkSolverApp.mlapp`
- 求解器：`General Problem Solver/+gps/+logic/ventilation_network_solver_generic.m`
- 回路识别：`General Problem Solver/+gps/+logic/identify_fundamental_loops.m`

## 2) 专用求解器（Solver for fig5.2）

适用：教材图 5.2 固定拓扑场景。

核心特性：
- 回路矩阵等结构固定/硬编码，单次求解速度更快；
- 适合教学/作业与参数扫描。

入口与说明：
- `Solver for fig5.2/README.md`
