# 02 项目目录与模块划分

本章给出目录结构与模块职责，便于你快速定位“数据在哪、算法在哪、UI 在哪”。

---

## 1. 顶层目录

- `README.md`：仓库总览与下载/安装说明
- `General Problem Solver/`：通用求解器（App + `gps.*` 包）
- `Solver for fig5.2/`：图 5.2 专用求解器
- `LICENSE`：GPL-3.0 许可证
- `document/`：本套整理后的文档（你正在阅读）

---

## 2. General Problem Solver 目录结构

核心采用 MATLAB 包（package）组织，命名空间为 `gps`：

- `General Problem Solver/ui/`
  - `NetworkSolverApp.mlapp`：App Designer 主界面
  - `NetworkSolverApp_exported*.m`：导出的 App 类文件（便于版本管理/对比）
- `General Problem Solver/+gps/+logic/`（核心算法）
  - `ventilation_network_solver_generic.m`：Hardy Cross 通用求解器
  - `identify_fundamental_loops.m`：基本回路自动识别（生成树法）
- `General Problem Solver/+gps/+data/`（离线数据加载/模板）
  - `load_network_data.m`：按目录读取 `branches.csv/boundary.csv/solver_config.csv`
  - `*_template.csv`：CSV 模板文件
- `General Problem Solver/+gps/+ui/`（UI 辅助/导入导出/绘图）
  - `solve_network_from_ui.m`：从 App 组件读取数据并调用求解器
  - `plot_network_graph.m`：绘制拓扑图（支持入/回风“边界巷道”显示）
  - `plot_solution_bars.m`：结果柱状图
  - `import_*` / `export_*`：CSV 导入导出
  - `add_new_row_to_uitable.m` / `delete_selected_rows_from_uitable.m` / `clear_uitable.m`：表格操作
